Configuration Parameters

This page documents all variables defined in General_Data.xlsx.

Each parameter controls a specific aspect of HydroPol2D, including:

  • simulation timing
  • numerical stability
  • hydrological processes
  • hydraulic routing
  • input datasets
  • output control

River Geometry Parameters​

If flag_reduce_dem is activated, river cells elevation are reduced to account for river bathymetry. A set of 4 parameters are then required to describe the flow width and depth. These parameters are valid to convert flow accumulation area [km2\mathrm{km^2}km2] into flow depth [m\mathrm{m}m] and width [m\mathrm{m}m].

VariableUnitsDescription
α1\alpha_1α1​—Empirical coefficient controlling channel geometry scaling
α2\alpha_2α2​—Secondary coefficient for hydraulic geometry relationships
β1\beta_1β1​—Exponent controlling width or hydraulic scaling behavior
β2\beta_2β2​—Secondary exponent for hydraulic scaling relationships
nnns m−1/3\mathrm{s\,m^{-1/3}}sm−1/3Manning’s roughness coefficient for channel and surface flow
